# Data Science Dashboard

A web-based dashboard for data analysis and machine learning.

## Features
- Data Upload (CSV, Excel, JSON)
- Clustering Analysis (K-Means, DBSCAN)
- Outlier Detection (Z-Score, IQR, Isolation Forest)
- Regression Analysis
- Apriori Algorithm
- Interactive Visualizations

## Quick Start
1. Install: `pip install flask pandas numpy scikit-learn openpyxl`
2. Run: `python app.py`
3. Open: `dashboard.html` in browser

## API
- Backend runs on: http://localhost:5000
- Frontend: Open dashboard.html
